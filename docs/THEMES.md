# Werm UI Themes

## Available Themes

### White (Default)
- Clean, modern light theme
- High contrast text
- Semi-transparent elements

### Black
- Dark theme with gray accents
- Easy on the eyes
- Professional look

### Purple  
- Purple color scheme
- Great for premium scripts
- Unique appearance

### Red
- Red accent theme
- High visibility
- Bold appearance

### Yellow
- Yellow/gold theme
- Warm color scheme
- Good readability

### Light  
- Original light theme
- Bright colors
- Classic look

### Default
- Original dark theme
- Familiar interface
- Traditional styling

## Changing Themes

To change the default theme, modify this line in the source:

```lua
local SelectedTheme = WermUILibrary.Theme.White -- Change to .Black, .Purple, etc.
